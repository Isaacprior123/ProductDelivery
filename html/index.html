API Endpoints



Get guild products (GET)

https://baseurl:3000/products/guild/

Request Body:

guildid: id

Response:

{ "productname": { "name": "productname", "description": "This is a description", "devid": "developerproductid" } }



Get user products (GET)

https://baseurl:3000/products/user/

Request Body:

robloxid: id guildid: id

Response:

["product1", "product2"]



Get Discord information (GET)

https://baseurl:3000/information/

Request Body:

robloxid: id

Response:

{ "id": "123456789", "username": "testusername", "descrim": "1234", "tag": "testusername#1234", "avatar": "https://url" }



Create purchase (POST)

https://baseurl:3000/purchase/

Request Body:

token: string guildid: id robloxid: id productname: name

Response:

{ "status": "ok", "dmerror": "false" }
